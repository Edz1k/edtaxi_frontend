# Картинки тарифов

Положи сюда изображения машин по одному файлу на тариф. Лучше **PNG с прозрачным
фоном** (панель тарифов тёмная — светлый фон будет выглядеть заплаткой). Также
можно `.webp`.

Имя файла = категория тарифа. Компонент `TariffStage.vue` подхватывает их
автоматически через `import.meta.glob`. Если файла нет — рисуется иконка-фолбэк.

| Файл           | Тариф   | Машина          |
| -------------- | ------- | --------------- |
| `economy.png`  | Эконом  | жёлтая          |
| `comfort.png`  | Комфорт | белая           |
| `business.png` | Бизнес  | чёрная          |
| `moto.png`     | Мото    | жёлтый мотоцикл |
| `minivan.png`  | Минивэн | жёлтый минивэн  |

Рекомендация по размеру: ~400×300 px, машина по центру, лишние поля обрезаны.
